# WhatsPilot AI

Day 36

Backend only.

## Features

- Uses Gemini API
- Generates WhatsApp replies
- Backend separated into modules
- No UI

## Run

Install packages

```bash
pip install -r backend/requirements.txt
```

Create `.env`

```env
GOOGLE_API_KEY=YOUR_KEY
```

Run

```bash
cd backend
python test.py
```
